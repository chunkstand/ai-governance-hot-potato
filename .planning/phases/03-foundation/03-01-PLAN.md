<task type="auto">
  <name>Task 3: Create PostgreSQL database schema with Prisma</name>
  <files>backend/prisma/schema.prisma, backend/src/lib/prisma.ts</files>
  <action>
    Create PostgreSQL database schema using Prisma ORM per requirement INF-02:
    
    1. Create backend/prisma/schema.prisma with models:
       
       model GameSession {
         id          String   @id @default(uuid())
         status      String   // INIT, AWAITING_ANSWERS, PROCESSING, RESOLVED, FINISHED
         createdAt   DateTime @default(now())
         updatedAt   DateTime @updatedAt
         finishedAt  DateTime?
         agents      Agent[]
         moves       Move[]
       }
       
       model Agent {
         id            String   @id @default(uuid())
         name          String
         type          String   // AI, HUMAN, DEMO_STRICT, DEMO_LENIENT, DEMO_BALANCED
         color         String   // hex color for UI
         position      Int      @default(0)
         score         Int      @default(0)
         gameSessionId String
         gameSession   GameSession @relation(fields: [gameSessionId], references: [id])
         decisions     Decision[]
         moves         Move[]
         createdAt     DateTime @default(now())
       }
       
       model Decision {
         id          String   @id @default(uuid())
         questionId  String
         answer      String   // A, B, C, D
         reasoning   String   @db.Text
         timeMs      Int      // time to answer in milliseconds
         isCorrect   Boolean?
         agentId     String
         agent       Agent    @relation(fields: [agentId], references: [id])
         createdAt   DateTime @default(now())
       }
       
       model Move {
         id          String   @id @default(uuid())
         fromPosition Int
         toPosition   Int
         spacesMoved  Int
         agentId     String
         agent       Agent    @relation(fields: [agentId], references: [id])
         gameSessionId String
         gameSession   GameSession @relation(fields: [gameSessionId], references: [id])
         createdAt   DateTime @default(now())
       }
    
    2. Create backend/src/lib/prisma.ts:
       - Import { PrismaClient } from '@prisma/client'
       - Create singleton prisma client instance
       - Export prisma for use in routes and services
       - Add connection error handling
    
    3. Add Prisma client generation to build process:
       - npm script: "db:generate": "prisma generate"
       - npm script: "db:migrate": "prisma migrate dev"
       - npm script: "db:deploy": "prisma migrate deploy"
    
    IMPORTANT: Use UUID for all IDs (not auto-increment) to avoid collision issues in distributed systems. Include @updatedAt for all models that track modification time.
  </action>
